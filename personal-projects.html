Web Scraps

I'd been coding a little under 3 years, a fancied learning a new language. Python seemed like the perfect candidate.

This was a fun side-project to build. You can use it to scrape text and json responses from around the web and download them as text files.

I decided to write my own databse abstraction layer (which, for simple crud, isn't exactly a chore), user and authenication flow is handled by flask sessions, passwords are hashed with bcrypt.

Technologies used: Python3, Flask, MySQL, SQLite3, Beautiful Soup, Bcrypt, Gunicorn wsgi, Heroku

UK coronavirus dashboard

This is what I did with all of my free time during the lockdown. I'd been learning about Angular and Slim PHP and wanted to put them to good use on a small project.

The api acts as gatway between the front-end, the NHS Conditions API and the RDBMS, exposing the statistics or widgets through endpoints. The front-end is completely decoupled and hosted on a seperate server running node.js. This serves compressed the compressed static files, forces https connections and routes all requests through the Angular entry point, the Index HTML file

Technologies used: Angular, Node.js, Chart.js, SlimPHP v3, MySQL, Heroku
